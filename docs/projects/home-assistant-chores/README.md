# Home Assistant Chores Dashboard

This project documents a Home Assistant dashboard for recurring household task visibility.

## Purpose

- Show active chores.
- Create recurring chores from Home Assistant.
- Display completed-today summaries.
- Keep the interface clean enough for everyday use.

## Current Status

- The chore backend is self-hosted.
- Home Assistant displays chore data.
- Daily chore creation works.
- Weekly chore creation has been added.
- Completion summaries are visible in Home Assistant.

## Design Notes

The backend handles recurring task data. Home Assistant provides the household-facing dashboard.

This project matters because it makes the infrastructure useful in daily life, not just technically interesting.
